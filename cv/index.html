About Me

Hi, my name is Ivan Savitsky and I develop web interfaces. I am based in Moscow, Russia, and currently working for AssistAgro on its Agritech Solutions platform.

I enjoy creating clean, performant, and accessible web interfaces with empathy to the end user as well as caring attitude towards business needs. I am proficient in web fundamentals — HTML, CSS, and JavaScript — and prefer building interfaces with Vue.js and its ecosystem tools.

Experience

AssistAgro

Front-end Developer

November 2021 – Present

assistagro.com

AssistAgro, an agritech startup, develops a complex SaaS platform, which incorporates drones, machine learning, predictive algorithms, satellite monitoring and meteo data to boost efficiency of agricultural producers.

Working in a team of 5 front-end developers on AssistAgro’s web interface, I deliver new features, maintain existing codebase, cover it with unit tests, and review code. The app is built with Nuxt.js and uses TypeScript as well as Composition API.

The image recognition interface, where we show ML-processed drone-taken images of fields, is built using canvas. Additionally, we utilize Mapbox to allow users to work with fields and monitor agricultural activities.

As we are building a complex interface with numerous forms and data visualizations, we maintain our custom UI kit and display it on Storybook for internal use.

Softlink has vast experience in producing professional investment software, such as trading platforms, financial robots, and automated investment services.

My primary task at Softlink was to develop and maintain the administrative panel interface of its leading automated investment advisor platform. The multi-language admin panel is built with Vue.js, Vue Router, Vuex, and Vuetify. I was working as a solo front-end developer in a team of 4 people and I took key front-end architecture decisions and carefully selected tools to build a convenient and performant web interface.

Finally, I developed the Softlink corporate website. It uses Strapi as a Node.js headless CMS, and Nuxt.js as a static pages generator. This setup blends flexible developer experience, convenient content management, and smooth end user experience.
